Jokenpô - Pedra, Papel e Tesoura
Descrição

Este é um jogo simples de Jokenpô (Pedra, Papel e Tesoura) implementado em C, onde o jogador enfrenta o computador.
O jogo sorteia a jogada do computador aleatoriamente e compara com a escolha do jogador para determinar o vencedor.

Como jogar
Execute o programa.
Escolha uma opção digitando o número correspondente:
1 para Pedra
2 para Papel
3 para Tesoura
O programa mostrará a jogada do jogador e do computador.
O resultado será exibido:
Vitória do jogador;

Vitória do computador;

Empate;
